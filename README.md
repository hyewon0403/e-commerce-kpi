## 이커머스 데이터 활용 지표 개발

### 프로젝트 소개
오픈마켓 런칭 이후 서비스 정체기로 인해 영업 이익과 사용 고객수가 감소하여 **서비스 사용 현황을 파악하고자 지표를 활용한 분석을 진행**했습니다.
최근성(recency), 구매 빈도(frequency), 구매 금액(monetary) 3가지 지표를 활용하여 고객 등급을 세분화하였고, 이를 바탕으로 시각화하여 인사이트를 추출하였습니다.
